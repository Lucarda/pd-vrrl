Port-forwarding




There are cases where you have to open your vrrl UDP port in your router. Thare are also cases where you can not do that but you can use a VPN.

Some links that covers port-forwarding:

https://www.noip.com/support/knowledgebase/general-port-forwarding-guide/

VPN:

If someone in the group has a linux machine it should be easy to setup a VPN using this script:

https://github.com/angristan/openvpn-install

that sets an OPENVPN server that can be used by the ones that cant do port-forwarding.
